Index of Python Material

    1. Our goal in Unit 1 is to make learners comfortable with Python by using it as a graphing calculator. In later units/volumes we will dive into more advanced numerical techniques. Below are the specific skills that we introduce in Unit 1 of Volume 1 and where they are introduced.

      In addition to these skills, there are Python examples that demonstrate how more advanced numerical techniques might be advantageous. The purpose of these are to be demonstrations rather than give the reader more to have to learn.

    1. Unit 1 Python Skills

      1. Variables, Arithmetic, and Plotting with Matplotlib (Using Computing Tools)
      2. Scientific notation (Section 1.2)
      3. Printing and f-strings (Section 1.6)
      4. Using NumPy arrays to represent vectors (Section 2.2)
      5. Using arrays to do vector algebra (Section 2.3)
      6. Computing dot products and magnitudes in Python (Section 2.4)
      7. Using functions to represent mathematical functions (3.1)
      8. Euler's method in 1 dimension.

    2. Additional Demonstrations in Unit 1

      1. Numerical Derivatives
      2. Euler's Method
      3. 3.2 Functions and numerical derivatives.
